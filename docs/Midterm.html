5) What did we do? (In LaTeX)

\[p_{ij}\] = probability tadpole(i) in tank(j) survives

Model Equation \[ logit(\rho_{ij})=\beta_{0}+\beta_{1}\cdot pred_{j}+\beta_{2}\cdot size_{j}+\beta_{3}\cdot (pred_{j}\times size_{j}) + u_{j} \]

\[u_{j}\] = random intercept for each tank(j) \[\sigma^{2}_{tank}\] = variance among tanks

Random Effect \[ u_{j} \sim \mathcal{N}(0, \sigma^{2}_{tank}) \]
